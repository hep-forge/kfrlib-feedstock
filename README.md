# kfrlib-feedstock

[![Build AMD64](https://github.com/hep-forge/kfrlib-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/kfrlib-feedstock/actions/workflows/autoupload.amd64.yml)
[![Build ARM64](https://github.com/hep-forge/kfrlib-feedstock/actions/workflows/autoupload.arm64.yml/badge.svg)](https://github.com/hep-forge/kfrlib-feedstock/actions/workflows/autoupload.arm64.yml)

Feedstock for [kfrlib](https://www.kfrlib.com) — part of [hep-forge](https://anaconda.org/hep-forge).

Fast, modern C++ DSP framework, FFT, Sample Rate Conversion, FIR/IIR/Biquad Filters (SSE, AVX, AVX-512, ARM NEON)

## Install

```bash
conda install -c hep-forge -c conda-forge kfrlib
```
